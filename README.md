# Description

This app was bootstrapped using [Expo](https://docs.expo.dev/guides/typescript/).

## Run app locally

- Run `npm install`
- Run `npx expo start`
- install the expo go app in your phone
- scan build QR code

## Run iOS locally
```
npx expo install expo-dev-client
npx expo run ios
```

## Deploy iOS/Android
```
npm install -g eas-cli
eas build
```
